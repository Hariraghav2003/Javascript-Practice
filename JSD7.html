JavaScript Arrays

Array is the collection of strings or integers or multiple data types as a single value we can access those elements using their index value and using multiple array methods which is based on the action which we are going to perform some common methods are

  • to convert array into string we use toString() method
  • to check the length of the array we use length function
  • to sort the array in asecnding order we use Sort method
  • to print the array we can use the array variable for printing the whole array for particular value we use the index value of the element

JS Array Methods

Basic JavaScript methods are,

  1. at():To access a particular element of an array using index (negative index values are also allowed )
  2. join():To join an element in an array
  3. pop():delete the first element of an arrays and prints that poped element
  4. push():adds the element at the end of an array
  5. shift():removes the first array element
  6. unshift():adds the element at the begining of the array
  7. delete:deletes the particular element in array using their index values but doesn't the allocated memory of that element which means it returns undefined
  8. concat():to join two arrays
  9. copywithin():To copy a element as a replace of another element we can copy a single value or multiple values
  10. flat():Converts a nested arrays into a single array if we want to use a function within this method we use flatmap()
  11. Splice():used to add a new element into an array this action will be performed in the original array if we orginal wants to be kept we can use tospliced()
  12. Slice():to cut out a elements from an array
