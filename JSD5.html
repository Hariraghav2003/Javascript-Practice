JavaScript Strings and methods!!

String are the collection of characters or combination of characters and there are many methods to access these Strings they are,

  1. String.length:used to find the length of the given string
  2. Sting.charAt():Used to extract the character from the string based on the index value
  3. String.charCodeAt():Same as charAt() but it returns that character's unicode value
  4. String.at():Same as charAt() but it allows negative index values i.e from last
  5. String[]:Same as charAt() but if the character at the given postion is not found it will return undefined whereas charAt returns empty string
  6. String.slice(Start,End):used to extract the part of the string based on index value given in the start and end but the character in the end value will not be included
  7. String.substring(Start,End):same as slice but negative index values are considerd as 0
  8. String.substr(Start,length):same as slice but the endvalue will be length if we give 1 as start value it will return the characters upto the length value if length value is not given it will take upto end of the string (This functionality is applicable to slice())
  9. String.touppercase():To convert the string to uppercase
  10. String.tolowercase():To convert the string to lowercase
  11. String.concat():To merge two strings
  12. String.trim():To remove the whitespace
  13. String.trimStart():To remove the whitespace before the text
  14. String.trimEnd():To remove the whitespace after the text
  15. String.padStart():To concate the values based on the index before text
  16. String.padEnd():To concate the values based on the index after text
  17. String.repeat():To repeat a string based on the value
  18. String.replace():To replace single characters in the given string based on the given text inside the replace it's case senstive so we use /i
  19. String.replaceAll(): Same as replace but it will replace All the occurance
  20. String.split():To convert string to character array based on " Comma (,) " spaces and pipes
