ArrayIteration Methods

Array Iteration Method are used to access each any element of the array or perform any operation or function on it

  1. foreach(): to display or to iterate over each element
  2. map():to perform function on each element
  3. flatmap():Converts the sub array into single array and performs function overit
  4. filter():To sort the elementss in an array based on a condition
  5. reduce():It's same as foreach but it takes an addtional parameter total
  6. reduceRight():Same as reduce() but it works from right to left
  7. every():Same as filter() but it all the elements should satisfy the contents and returns boolean values
  8. some():Same as every() but any one of the elements should satisfy the condition
  9. from():to convert string to array
  10. keys(): returns the indexes of array
  11. entries():returns the key value pair
  12. with():to update the element of the array using index value
  13. spred(...):To print multiple arrays in one line
