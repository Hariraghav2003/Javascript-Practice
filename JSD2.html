Invocation function

Invocation function is calling a function using objects or directly writting the function like this There are six different ways to invoke a function and they are

  1. Direct Invocation:calling the function by it's name
  2. Method Invocation:calling the function through object
  3. Constructor Invocation:calling the function through object but the object and function name will be same
  4. Indirect Invocation:calling the function indirectly using call, apply, and bind
  5. Immediately Invoked Function Expression:calling the function Immediately
  6. Arrow Function Invocation:declaring functions as Expressions using arrows

Indirect Invocation

indirect Invocation is calling the function indirectly using call,apply and bind keyword usage of this keyword will differs based on the variable which we are going to use for object variables we use call the example code,

{
let sum=a+b;
document.getElementById("iic").innerHTML=sum;
} let c={a:40,b:50};
indirectinvocationc.call(c,c.a,c.b);

Here the c is a object variable so we use call keyword to call the function indirectinvocationc

for array variables we use apply the example code,

{

Arrow function

Arrow function is declaring functions as Expressions like this

const add = (a, b) => a + b;
document.getElementById("af").innerHTML=add(8,8);

Including this there are some more functions and they are

  1. Anonymous Function:These functions are passed as a arguments or declared in a variable
  2. Callback Function:These functions are passed as a arguments to another and then they are called in that function
  3. Generator function:The Generator function is a recursive function
  4. Async function:Async function waits and check for the requirements and then runs the function declared in the await
