String Search Methods & Integers

String search methods are used to search a particular letter or word in a specifed string based on their index values or based on values

  1. String.indexOf(): is used to search a string based on their index values and returns the first occurance of the given string and also we can customize the starting index if the value is found it returns index of the string
  2. Sting.lastIndexof():same as firstindexof() but it return the last occurance of the given string
  3. String.search(): is used tosearch the string based on their values in this method we can't customize the starting index if the value is found it returns 0 if not found returns -1
  4. String.match():returns the first occurance of the string
  5. String.matchAll():returns the all occurance of the string
  6. String.includes():Returns whether the given string is present or not in that string
  7. String.startswith():Returns whether the given string starts with a particular value
  8. String.startswith():Returns whether the given string ends with a particular value
All these methods are case senstive so if we need to search irrespectiv of the case we can use /i

We can also convert the strings to numbers using parseInt or number to string using tostring() to check the type we use typeof() method
