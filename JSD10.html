Control Statements

Javascript control statements are If..else,else if,Switch statements

  • in If..else statement there will be a condtion in if block if the condtion is true it will execute the part or if it's false the code define in the else part will be excuted
  • else if statement contains the second condtion if the condtion in the if block is false then it will check the condtion in the else if and if it's true it will be executed
  • Switch statements are same as if statements but it contains three keywords case,default,break. The case contains the code which is wanted to be executed,break is if any one of the case is executed then other part of code will not be executed default is like else part

Looping Statements

Looping statements are used to iterate over the elements which is of 5 types:for loop,for..of,for..in,while,do..while

  • forloop:used to iterate over the elements based on condtion and has iterable part
  • for..of:used to diplay the javascript array
  • for..in:used to display the object elements
  • while:used to iterate over the element based on condtion but it doesn't have any iterables
  • do..while:same as while but the do part is always executed

Break

break statements are continue,jump

  • continue is used to execute the code
  • break is used to stop the code at a point
