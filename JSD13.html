What is DOM??

The JavaScript Document Object Model through which we can access all the HTML CSS properties there two different DOM, ACtual DOM,Virtual DOM if there is a 4 webpages and we change the one page properties The Actual DOM will referesh all the 4 pages where Virtual DOM refereshes the particular page

DOM Methods

The JavaScript DOM has many methods to access the HTMl elements some come methods are,

  • document.getelementid():To get the HTML element through their ID
  • document.getelementbyTagname():To get the HTML element through their tagname
