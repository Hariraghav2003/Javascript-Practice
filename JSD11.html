JS Sets

JS sets are similar to Array and object but the difference is sets don't allow duplicates it has 6 methods,

  1. .add():to add a new element
  2. .has():to search for a particular element
  3. .foreach();to display all the elements in the set
  4. .values():to return the values in the set
  5. .keys():to return keyvalues in the set but set has no key values so the value will be it's key
  6. .enteries():to return key value pair

JS Map

JS Maps are similar to Set but it allows duplicates to avoid it we can pass set into map by using set() method and has 4 addtional methods,

  1. .get():to get a particular element from the map
  2. .clear():to delete the entire map
  3. .delete():to delete a particular element in a map
  4. .size:to check the size of the map

JS Closure

JavaScript Closure function has the access to the variable which is outside it's block even after it's process is completed function Closure(){
let c="Hello I am a variable outside the cl block";
function cl(){
console.log(c);
}
cl();
}
Closure();
Here cl() is the closure function since it has access to the parent variable c even it finishes the execution.
