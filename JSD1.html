Importing Javascript

Javascript is used to define the behaviour of the website which can be imported into html file using two ways,

  • Internal JS:where the Javascript is written inside the html file using Script Tag
  • External JS:where the Javascript is written in a seperate file in .js extension and linked into html file using script tag
The version of Javascript is ES2015 (Egma Script)

Javascript Output

The output of the Javascript is displayed using 4 different ways

  1. document.getElementById.innerHTML:the output is displayed by passing the content to elements through ID
  2. document.write():Used to display the contents inside the function
  3. window.alert():used to pop up a alert box
  4. console.log():Used for testing purpose if the contents inside this function runs smoothly then we implement it in real time

JS Variables

Variables are the containers used to store the data there are 4 ways to declare a Variable in Javascript and they are

  • Var: is a keyword to declare a Variable
  • Let:is the advanced version of Var
  • const:is the keyword used to declare the universal values such as pie the data stored inside the Variables declared using this keyword can't be changed
  • Auto:without using any keywords we can declare Variables like this x=0;
