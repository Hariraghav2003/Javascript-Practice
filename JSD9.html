const date=new Date(); where Date() has these arguments year,month,day,hours,minutes,seconds,milliseconds.

JavaScript Math

JavaScript math functions are used to perform math function in JavaScript where there are 3 functions,

  • Math.round():returns the integer by rounding off the decimal value if 4.7 then round(4.7) will be 5
  • Math.ceil():returns the integer by rounding off the decimal value if 4.7 then ceil(4.7) will be 4
  • Math.floor():returns the nearest integer value
